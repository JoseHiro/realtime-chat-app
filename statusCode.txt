| コード | 意味                  | 説明                  |
| --- | ------------------- | ------------------- |
| 100 | Continue            | クライアントはリクエストを継続してよい |
| 101 | Switching Protocols | プロトコル切替を要求された       |

| コード | 意味         | 説明                          |
| --- | ---------- | --------------------------- |
| 200 | OK         | リクエスト成功（GET, PUT, PATCH など） |
| 201 | Created    | リソースが作成された（POST）            |
| 202 | Accepted   | リクエストは受け付けられたが処理はまだ完了していない  |
| 204 | No Content | 処理成功だがレスポンスボディなし            |

| コード | 意味                         | 説明              |
| --- | -------------------------- | --------------- |
| 301 | Moved Permanently          | 永久にリソースが移動      |
| 302 | Found / Temporary Redirect | 一時的にリソースが移動     |
| 304 | Not Modified               | キャッシュを使用可能、更新なし |

| コード | 意味                   | 説明                 |
| --- | -------------------- | ------------------ |
| 400 | Bad Request          | リクエストが不正           |
| 401 | Unauthorized         | 認証が必要              |
| 403 | Forbidden            | アクセス権なし            |
| 404 | Not Found            | リソースが存在しない         |
| 405 | Method Not Allowed   | 許可されていない HTTP メソッド |
| 409 | Conflict             | データの競合（例: 重複登録）    |
| 422 | Unprocessable Entity | バリデーションエラー         |

| コード | 意味                    | 説明               |
| --- | --------------------- | ---------------- |
| 500 | Internal Server Error | サーバー内部エラー        |
| 502 | Bad Gateway           | ゲートウェイ経由でのエラー    |
| 503 | Service Unavailable   | サービス停止中、メンテナンスなど |
| 504 | Gateway Timeout       | ゲートウェイがタイムアウト    |

成功系: 200 系

エラー系: 400（クライアント） / 500（サーバー）

作成: POST の後は 201 が推奨

更新: PUT/PATCH は 200 または 204
